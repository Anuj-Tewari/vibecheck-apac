VibeCheck APAC 🔥
The Cultural Sentiment Intelligence Agent

VibeCheck is an AI Agent built for the GenAI Academy APAC Edition. It bridges cultural communication gaps by decoding regional slang and sentiment that standard translation tools often miss.

🚀 Key Features
Slang Analysis: Intelligent decoding of regional terms like 'Lodi', 'Mantap', and 'Jugaad'.

Vibe Scoring: A 1-10 quantitative metric to measure social resonance.

Cultural Context: Real-time etiquette notes for India, Indonesia, and the Philippines.

Visual Feedback: Gamified UI with celebratory balloons for high-vibe scores.

🛠️ Tech Stack
LLM: Gemini 1.5 Flash (via Agentic Workflow)

Frontend: Streamlit

Backend: Flask API

Deployment: Google Cloud Run (Serverless)
